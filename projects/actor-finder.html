One-time setup

This page talks directly to The Movie Database. It needs your personal (free) API key, which is saved only in this browser — never in the repository.

  1. Create a free account at themoviedb.org
  2. Go to Settings → API, request a key (type: Developer)
  3. Copy the value labelled API Key (v3 auth) and paste it here
